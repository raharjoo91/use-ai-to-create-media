## Your First AI-Powered K3 Poster (15 Minutes)

### Step 1: Research with NotebookLM (5 minutes)

```
1. Go to: https://notebooklm.google.com
2. Click "Create Notebook"
3. Name it: "First K3 Poster"
4. Add Source: Upload a K3 PDF or paste a website URL
   (Try: WHO workplace health guidelines)
5. Wait for processing (~1 minute)
6. Type in chat: "Summarize key safety points for office workers"
7. Copy the key points generated
```

**Output:** 3-5 key safety points ready for your poster

### Step 2: Design with Canva AI (8 minutes)

```
1. Go to: https://canva.com
2. Click "Create a design" → "Custom size"
3. Enter: 24 x 36 inches (standard poster)
4. Type: "/" for shortcuts → "Magic Design"
5. Enter prompt: "Safety poster for office workers"
6. Choose a generated template you like
7. Replace the text with your NotebookLM key points:
   - Headline: Main safety message
   - Body: Your key points
   - Call to action: What to do
8. Adjust colors using ISO 7010:
   - Red: Danger/prohibition (#E74C3C)
   - Yellow: Caution (#F39C12)
   - Green: Safe/exit (#27AE60)
   - Blue: Mandatory (#1A5F7A)
9. Click "Download" → "PDF Print"
```

**Output:** Your first AI-powered K3 poster!

### Step 3: Review and Deploy (2 minutes)

```
Before Printing:
☐ View at 100% zoom
☐ Check font sizes are readable
☐ Verify colors match safety standards
☐ Confirm message is clear
☐ Test with 1-2 colleagues

After Printing:
☐ Place at eye level
☐ Ensure good lighting
☐ Note feedback from workers
☐ Plan improvements for next version
```

## Troubleshooting

### "NotebookLM won't accept my file"

**Solutions:**
- Check file is under 50MB
- Ensure it's PDF, Google Doc, or text
- Try copying text and pasting instead
- Use website URL for public sources

### "Canva template looks nothing like I want"

**Solutions:**
- Be more specific in your prompt
- Add style keywords: "minimalist", "bold", "professional"
- Try different prompts and compare
- Start with blank canvas and build from elements

### "Text doesn't fit on the poster"

**Solutions:**
- Reduce to 3-5 key points only
- Increase font size by 20%
- Remove decorative elements
- Create second poster for remaining info

### "Colors don't look right"

**Solutions:**
- Use exact hex codes from our color palette
- Check contrast with WebAIM checker
- Test in grayscale for accessibility
- Ask colleagues for feedback

## Quick Reference Cards

### Font Sizes by Distance

```
Viewing Distance → Minimum Font Size
1 meter         → 18 pt
3 meters        → 36 pt
5 meters        → 60 pt
10 meters       → 120 pt
```

### Safety Colors (ISO 7010)

```
Red (#E74C3C)    = Danger, Stop, Fire
Yellow (#F39C12) = Caution, Warning
Green (#27AE60)  = Safe, Exit, First Aid
Blue (#1A5F7A)   = Mandatory Action
```

### Reading Level

```
Target: Grade 6-8
Check: Use Flesch Reading Ease test
Score: 60-70 is good
```
