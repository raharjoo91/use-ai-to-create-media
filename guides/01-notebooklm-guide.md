## What is NotebookLM?

**NotebookLM** is Google's AI-powered research assistant that helps you:
- Understand complex documents
- Generate summaries and FAQs
- Create audio overviews
- Track citations to sources

### Why Use NotebookLM for K3?

| Traditional Approach | NotebookLM Approach |
|---------------------|---------------------|
| Hours reading PDFs | Minutes to synthesis |
| Manual note-taking | Auto-generated summaries |
| Lost citations | Source tracking built-in |
| Static content | Interactive Q&A |

## Getting Started

### Step 1: Access NotebookLM

1. Go to [notebooklm.google.com](https://notebooklm.google.com)
2. Sign in with your Google Account
3. Create your first notebook

### Step 2: Prepare Your K3 Sources

Gather relevant documents:
- ✅ WHO guidelines
- ✅ ILO conventions
- ✅ ISO standards
- ✅ Local regulations (Permenaker RI)
- ✅ Company policies
- ✅ Research papers

**Supported Formats:**
- PDF (up to 50MB)
- Google Docs
- Google Slides
- Websites
- Text files

## Upload & Organize Sources

### Method 1: Single File Upload

```
1. Click "Add Source"
2. Select "Google Drive" or "Upload from computer"
3. Choose your K3 PDF/document
4. Wait for processing (usually < 1 minute)
5. Review the source summary
```

### Method 2: Multiple Related Sources

For comprehensive K3 topics:

```
Fire Safety Notebook
├── ISO 7010 Standard (fire signs)
├── Permenaker No. 4 (fire safety)
├── Company Emergency Plan
└── Training slides
```

### Method 3: Website Sources

```
1. Click "Add Source"
2. Select "Website"
3. Paste URL (e.g., https://www.ilo.org/safety)
4. Add source
```

**Best K3 Websites to Source:**
- ILO: ilo.org/safetywork
- WHO: who.int/occupational-health
- OSHA: osha.gov
- Kemenaker RI: kemnaker.go.id

## Generate K3 Content

### 1. Generate Summary

**Prompt:**
```
Summarize the key fire safety procedures for office buildings
```

**Output:**
- Main points extracted
- Citations linked to sources
- Actionable items highlighted

### 2. Create FAQ

**Prompt:**
```
Create 10 frequently asked questions about PPE usage in construction
with answers based on the uploaded documents
```

**Use FAQs for:**
- Poster content
- Training handouts
- Quick reference guides

### 3. Generate Audio Overview

```
1. Click "Notebook Guide" → "Generate"
2. Wait 2-3 minutes
3. Listen to 5-10 minute overview
4. Use as background for video narration
```

**Audio Overview Uses:**
- Training podcast
- Commute learning
- Accessibility for visual impairments

### 4. Q&A Mode

**Ask specific questions:**
```
Q: What are the color codes for safety signs according to ISO 7010?
A: [NotebookLM provides answer with citations]
```

## Best Practices

### 1. Source Organization

**DO:**
- Group related documents
- Use descriptive notebook names
- Remove outdated sources
- Keep sources < 20 per notebook

**DON'T:**
- Mix unrelated topics
- Upload poor-quality scans
- Exceed 50MB file limit
- Forget to update sources

### 2. Prompt Engineering

| Poor Prompt | Better Prompt |
|-------------|---------------|
| "Tell me about safety" | "List 5 key electrical safety procedures for office workers" |
| "Make content" | "Create a 300-word summary of hazard identification steps" |
| "Give me everything" | "Extract the 3 most important points from the emergency protocol" |

### 3. Citation Verification

**Always:**
1. Click citation links
2. Verify accuracy
3. Cross-reference with original
4. Update if guidelines change

## Common Issues

### Issue 1: "Source Not Found"

**Cause:** File was moved or deleted
**Fix:** Re-upload the source to your notebook

### Issue 2: "Generic Response"

**Cause:** Prompt too vague
**Fix:** Be specific about:
- Topic scope
- Target audience
- Output format
- Length constraints

### Issue 3: "Outdated Information"

**Cause:** Old source documents
**Fix:**
1. Check publication dates
2. Upload newer versions
3. Remove outdated sources
4. Add disclaimer for date-sensitive content

### Issue 4: File Size Exceeded

**Cause:** PDF > 50MB
**Fix:**
- Split into smaller files
- Compress images
- Extract text only
- Use website source instead

## K3 Use Cases

### Use Case 1: Poster Content Development

**Goal:** Create poster about chemical handling

**Workflow:**
```
1. Upload chemical safety PDFs
2. Prompt: "Extract 5 key handling precautions for corrosive chemicals"
3. Review and verify points
4. Use in Canva poster design
```

**Output:**
- ✅ Clear, concise points
- ✅ Cited to reliable sources
- ✅ Ready for design phase

### Use Case 2: Training Material Creation

**Goal:** Create handout for new employee safety orientation

**Workflow:**
```
1. Upload orientation manual + regulations
2. Prompt: "Create FAQ for new workers about emergency procedures"
3. Generate 10 Q&A pairs
4. Format for printing
5. Add to training packet
```

### Use Case 3: Policy Update Summary

**Goal:** Summarize new regulation changes

**Workflow:**
```
1. Upload old and new regulation versions
2. Prompt: "Identify 3 key changes between these documents"
3. Generate comparison
4. Create communication to staff
```

### Use Case 4: Video Script Research

**Goal:** Research for Sora video about ergonomics

**Workflow:**
```
1. Upload ergonomics guidelines + research
2. Prompt: "List common workstation hazards and solutions"
3. Generate audio overview
4. Use points + audio as script foundation
```
