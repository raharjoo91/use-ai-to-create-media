## What is Canva AI?

Canva AI combines multiple features to automate and enhance design:

| Feature | What It Does | K3 Application |
|---------|--------------|----------------|
| **Magic Design** | Auto-generates layouts | Quick poster creation |
| **Text to Image** | Creates images from prompts | Custom safety graphics |
| **Magic Edit** | Edits image elements | Adapting stock photos |
| **Magic Write** | Generates copy | Poster text creation |
| **Brand Kit** | Ensures consistency | Corporate K3 branding |

## K3 Design Fundamentals

### Font Size by Distance

| Viewing Distance | Headline | Body Text |
|------------------|----------|-----------|
| 1-2 m (leaflet) | 48-60 pt | 18-24 pt |
| 3-5 m (A2 poster) | 72-96 pt | 24-36 pt |
| 5-10 m (A1 poster) | 120-180 pt | 36-48 pt |
| 10+ m (banner) | 200+ pt | 60+ pt |

**Rule of Thumb:** Every 10 feet = double the text size

### ISO 7010 Safety Colors

```
RED (E74C3C)    = Stop, Danger, Fire Prohibited
YELLOW (F39C12) = Caution, Warning, Hazard
GREEN (27AE60)  = Safe, Go, First Aid, Exit
BLUE (1A5F7A)   = Mandatory Action, Information
```

### Visual Hierarchy

```
1. HEADLINE (largest, most prominent)
   2. Key Image (supports headline)
   3. Supporting Points (bullet list)
   4. Call to Action (what to do)
   5. Contact Info (where to get help)
```

## AI Tools Overview

### 1. Magic Design

**Best for:** Quick poster creation from scratch

**How to Use:**
```
1. Open Canva → "Create a design"
2. Select custom size or template
3. Type prompt: "Fire safety poster for office building"
4. Choose generated option
5. Customize with your content
```

**Effective Prompts:**
- "Construction safety poster about PPE helmets"
- "Office ergonomics infographic showing proper desk setup"
- "Chemical handling poster with hazard symbols"

### 2. Text to Image

**Best for:** Custom safety graphics when stock photos don't fit

**How to Use:**
```
1. Click "Elements" → "Images" → "Magic Media"
2. Enter prompt describing image
3. Wait 10-30 seconds
4. Select best option
5. Add to design
```

**Prompt Tips:**
```
Good: "Worker wearing yellow hard hat and blue safety vest,
      adjusting safety harness on construction site,
      professional photo style, bright lighting"

Bad: "Construction worker safety"
```

### 3. Magic Edit

**Best for:** Modifying stock images to match your workplace

**How to Use:**
```
1. Select image in design
2. Click "Edit Photo" → "Magic Edit"
3. Brush over area to change
4. Describe replacement
5. Generate and apply
```

**Use Cases:**
- Change PPE color to match your company
- Add your company logo to uniforms
- Modify background to match workplace
- Remove distracting elements

### 4. Magic Write

**Best for:** Copywriting assistance

**How to Use:**
```
1. Add text box to design
2. Type "Press "/" for shortcuts
3. Select "Magic Write"
4. Enter prompt
5. Insert generated text
```

**Effective Prompts:**
- "Write 5 bullet points about ladder safety"
- "Create catchy headline for hand washing poster"
- "Generate emergency contact section"

## Templates & Customization

### Template Structure for K3 Posters

```
┌─────────────────────────────────────┐
│         HEADLINE (RED)              │ ← Zone 1: Above fold
│  [Eye-level, most important]        │
├─────────────────────────────────────┤
│  [Key Image]        │  [Bullet      │ ← Zone 2: Main content
│                      │   Points]     │
│  Visual supports     │               │
│  main message       │  • Point 1    │
│                      │  • Point 2    │
│                      │  • Point 3    │
├─────────────────────────────────────┤
│  [Call to Action]    │  [Contact]    │ ← Zone 3: Action
│  "Wear Your PPE!"    │  Safety x555  │
└─────────────────────────────────────┘
```

### Accessible Design Checklist

```
Color & Contrast:
☐ Color contrast ≥ 4.5:1 (use WebAIM checker)
☐ Not color-dependent for meaning
☐ ISO 7010 colors used correctly

Typography:
☐ Sans-serif font (Arial, Verdana)
☐ Font size appropriate for distance
☐ Line spacing 1.5x font size
☐ No text smaller than 24pt body

Layout:
☐ Clear visual hierarchy
☐ White space for breathing room
☐ Consistent alignment
☐ Reading order is obvious

Content:
☐ Language at grade 6-8 level
☐ Universal symbols included
☐ Jargon avoided
☐ Action clearly stated
```

## Export & Deployment

### Export Options

| Use Case | Format | Settings |
|----------|--------|----------|
| Professional Print | PDF Print | 300 DPI, CMYK |
| Office Print | PDF Standard | 150 DPI, RGB |
| Digital Display | PNG | 72 DPI, RGB |
| Video Editing | PNG | 300 DPI, transparent |
| Email/Web | JPG | 150 DPI, compressed |

### Print Preparation

```
Before Printing:
1. View at 100% zoom
2. Check all text is editable
3. Verify no low-resolution warnings
4. Confirm margins are safe
5. Test print one copy
6. Check colors are accurate
7. Verify from viewing distance
8. Order final quantity
```

### Digital Deployment

```
For Digital Signage:
1. Export as PNG (1920x1080)
2. Test on actual display
3. Check animation (if used)
4. Schedule rotation
5. Update regularly

For Email/SharePoint:
1. Export as PDF under 5MB
2. Test on mobile devices
3. Include alt text
4. Provide text description
```

## Common Issues & Solutions

### Issue 1: Text Too Small

**Symptom:** Can't read from intended distance
**Solution:**
```
1. Select all text
2. Increase by 20%
3. Reduce content to fit
4. Remember: Better too big than too small
```

### Issue 2: Low Contrast

**Symptom:** Hard to read, fails accessibility
**Solution:**
```
1. Use contrast checker: webaim.org/resources/contrastchecker/
2. Dark text on light background (or vice versa)
3. Avoid color-on-color of similar brightness
4. Test with grayscale view
```

### Issue 3: Too Much Information

**Symptom:** Poster looks cluttered
**Solution:**
```
1. Identify 1 main message
2. Remove everything else
3. Create QR code to detailed info
4. Use "less is more" principle
```

### Issue 4: Image Quality Poor

**Symptom:** Blurry when enlarged
**Solution:**
```
1. Use high-resolution originals (min 1000px)
2. Avoid enlarging small images
3. Use vector graphics when possible
4. Regenerate with Text to Image at higher scale
```

## Advanced Tips

### Tip 1: Batch Creation

```
Create multiple related posters at once:
1. Use consistent template
2. Generate all images in one session
3. Copy-paste common elements
4. Review as a set
5. Export all together
```

### Tip 2: Version Control

```
File naming convention:
poster_fire-safety_v1.0_draft_2026-04-05.pdf
poster_fire-safety_v1.1_final_2026-04-10.pdf

Keep all versions until final approval
```

### Tip 3: User Testing

```
Before final deployment:
1. Show to 3-5 workers
2. Ask: "What is this telling you to do?"
3. Note misunderstandings
4. Revise and retest
```
