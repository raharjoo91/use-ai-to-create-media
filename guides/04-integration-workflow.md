## Campaign Planning

### Step 1: Define Campaign Objectives

**Use SMART Framework:**

```
S - Specific: What exact K3 issue?
M - Measurable: What change expected?
A - Achievable: Realistic with resources?
R - Relevant: Linked to organizational goals?
T - Time-bound: When to see results?
```

**Example Campaign Brief:**

```
Campaign: "Heat Stress Prevention"
Target: Outdoor workers (construction, agriculture)
Objective: Reduce heat-related incidents by 50%
Duration: 3-month campaign (May-July)
Messages:
  1. Recognize heat stress symptoms
  2. Know prevention strategies
  3. Understand response procedures
Deliverables:
  - 3 posters (symptoms, prevention, response)
  - 1 infographic (risk assessment)
  - 2 videos (60-sec each)
  - 1 FAQ handout
```

### Step 2: Content Mapping

**Create Content Matrix:**

| Message | Format | Channel | Frequency | Call to Action |
|---------|--------|---------|-----------|----------------|
| Symptoms | Poster | Break room | Permanent | "Know the signs" |
| Prevention | Poster | Worksite | Permanent | "Stay hydrated" |
| Response | Poster | Emergency area | Permanent | "Act fast" |
| Risk Assessment | Infographic | Training handout | Monthly | "Check your risk" |
| Prevention Demo | Video | Toolbox talk | Weekly | "Watch and learn" |
| Emergency Response | Video | Training | Quarterly | "Be prepared" |

### Step 3: Asset Collection

**Gather Source Materials:**

## Production Pipeline

### Phase 1: Research & Content (NotebookLM)

**Workflow:**

```
1. CREATE NOTEBOOK
   ├─ Name: "Heat Stress Prevention Campaign"
   └─ Description: "Content for multi-format K3 campaign"

2. UPLOAD SOURCES (15-20 min)
   ├─ WHO guidelines PDF
   ├─ ILO standards PDF
   ├─ Company policy
   ├─ Research articles
   └─ Existing training materials

3. GENERATE CORE CONTENT (30-45 min)
   ├─ Prompt: "Summarize key heat stress symptoms"
   ├─ Prompt: "List prevention strategies for outdoor workers"
   ├─ Prompt: "Describe emergency response procedures"
   └─ Prompt: "Create 10 FAQs about heat stress"

4. CREATE AUDIO OVERVIEW (5 min wait, 10 min review)
   ├─ Use for video narration inspiration
   └─ Extract key phrases for posters

5. EXPORT CONTENT (10 min)
   ├─ Copy all generated text
   ├─ Organize by message/topic
   └─ Save for design phase
```

**Output Checklist:**
```
☐ Symptom list (3-5 key points)
☐ Prevention strategies (5-7 key points)
☐ Response procedures (step-by-step)
☐ FAQ document (10 Q&A pairs)
☐ Statistics/data for infographics
☐ Citation list for verification
```

### Phase 2: Design & Visuals (Canva AI)

**Workflow for Posters:**

```
1. SET UP BRAND KIT (one-time, 10 min)
   ├─ Upload logo
   ├─ Define colors (ISO 7010 compliant)
   ├─ Select fonts (sans-serif)
   └─ Save brand template

2. CREATE POSTER 1: SYMPTOMS (30 min)
   ├─ Magic Design: "Medical poster about heat stress symptoms"
   ├─ Replace with NotebookLM content
   ├─ Generate: Worker showing symptoms image
   ├─ Verify colors (red for danger)
   ├─ Check font sizes for distance
   └─ Export as PDF

3. CREATE POSTER 2: PREVENTION (30 min)
   ├─ Use template from Poster 1
   ├─ Replace with prevention strategies
   ├─ Generate: Hydration, shade, PPE images
   ├─ Verify colors (green for safe actions)
   ├─ Check readability
   └─ Export as PDF

4. CREATE POSTER 3: RESPONSE (30 min)
   ├─ Use template from Poster 1
   ├─ Replace with response procedures
   ├─ Generate: Emergency action images
   ├─ Verify colors (blue for mandatory)
   ├─ Check action steps clarity
   └─ Export as PDF

5. CREATE INFOGRAPHIC (45 min)
   ├─ Template: Data visualization
   ├─ Insert statistics from NotebookLM
   ├─ Create risk assessment flowchart
   ├─ Add icons for visual support
   ├─ Verify data accuracy
   └─ Export as PNG and PDF
```

**Workflow for Storyboards:**

```
1. CREATE VIDEO STORYBOARD (20 min)
   ├─ Template: Film/Video storyboard
   ├─ Outline scenes based on content
   ├─ Sketch visual concepts
   ├─ Note camera angles
   └─ Add timing notes

2. EXPORT FOR VIDEO PRODUCTION
   ├─ Save as PDF
   ├─ Export frames as images
   └─ Prepare prompt notes
```

### Phase 3: Video & Motion (Sora)

**Workflow for Video Production:**

```
1. PREPARE SCRIPTS (20 min)
   ├─ Use NotebookLM FAQ as dialogue
   ├─ Write narration script
   ├─ Plan visual sequences
   └─ Create prompt list

2. GENERATE VIDEO 1: PREVENTION DEMO (45 min)
   ├─ Scene 1: Worker hydrating
   │   └─ Prompt: "Construction worker in PPE drinking water
   │              at shaded rest area, professional training video"
   ├─ Scene 2: Taking breaks
   │   └─ Prompt: "Worker resting in shade, checking watch,
   │              following work-rest schedule"
   ├─ Scene 3: Wearing proper clothing
   │   └─ Prompt: "Worker in light-colored, loose-fitting
   │              clothing, adjusting hat for sun protection"
   ├─ Review and select best clips
   └─ Generate alternatives if needed

3. GENERATE VIDEO 2: EMERGENCY RESPONSE (45 min)
   ├─ Scene 1: Recognizing symptoms
   │   └─ Prompt: "Coworker noticing worker showing signs
   │              of heat exhaustion, concerned expression"
   ├─ Scene 2: Taking action
   │   └─ Prompt: "Worker guiding affected person to shade,
   │              offering water, radioing for help"
   ├─ Scene 3: Medical response
   │   └─ Prompt: "First responder checking vitals, applying
   │              cool compresses, professional medical care"
   ├─ Review and select best clips
   └─ Generate alternatives if needed

4. ASSEMBLE VIDEOS (30 min)
   ├─ Import clips to editor
   ├─ Arrange in sequence
   ├─ Add transitions
   ├─ Insert text overlays
   ├─ Add narration (AI or human)
   ├─ Include captions
   └─ Export final video
```

### Phase 4: Integration & Package

**Final Assembly:**

```
1. CREATE CAMPAIGN PACKAGE (20 min)
   ├─ Folder structure:
   │   ├─ Posters/ (PDF + source)
   │   ├─ Infographics/ (PNG + PDF)
   │   ├─ Videos/ (MP4)
   │   ├─ Handouts/ (PDF)
   │   └─ Deployment Guide/
   └─ Naming convention: "heat-stress_[type]_v1.0"

2. CREATE DEPLOYMENT GUIDE (30 min)
   ├─ Poster placement locations
   ├─ Video showing schedule
   ├─ Handout distribution plan
   ├─ Training integration points
   └─ Success metrics

3. CREATE FACILITATOR GUIDE (30 min)
   ├─ Talking points
   ├─ Discussion questions
   ├─ Quiz questions
   ├─ Feedback form
   └─ Evaluation checklist
```

## Quality Assurance

### QA Checklist

**Content Verification:**
```
☐ All information fact-checked
☐ Sources cited correctly
☐ Procedures are accurate
☐ Up-to-date regulations used
☐ No misleading claims
☐ Cultural appropriateness
☐ Language accessibility
```

**Design Review:**
```
☐ Brand consistency
☐ Color contrast ≥ 4.5:1
☐ Font sizes appropriate
☐ ISO 7010 colors correct
☐ Visual hierarchy clear
☐ No copyright violations
☐ Export quality correct
```

**Video Review:**
```
☐ Safety procedures accurate
☐ Realistic scenarios
☐ Clear audio
☐ Accurate captions
☐ Appropriate length
☐ Works on all devices
☐ Includes accessibility features
```

### Testing Protocol

**Alpha Test (Internal):**
```
Participants: 2-3 team members
Format: Review of all materials
Feedback: Design and content issues
Action: Make revisions
```

**Beta Test (Target Audience):**
```
Participants: 5-10 workers
Format: Mock deployment
Feedback: Comprehension and relevance
Action: Final refinements
```

**Pilot Test (Live):**
```
Location: One department/site
Duration: 1-2 weeks
Metrics: Awareness and behavior
Action: Organization-wide rollout
```

## Troubleshooting

### Common Campaign Issues

**Issue: Low Engagement**
```
Diagnosis: Content not relevant
Solution: Survey workers, address real concerns
Prevention: Involve target audience in planning
```

**Issue: Confusing Messages**
```
Diagnosis: Too much information
Solution: Simplify, focus on 1-3 key points
Prevention: Test with small group first
```

**Issue: Poor Visibility**
```
Diagnosis: Wrong placement
Solution: Use high-traffic areas, eye level
Prevention: Conduct site survey before deployment
```

**Issue: Outdated Information**
```
Diagnosis: Old sources or procedures changed
Solution: Update content, version control
Prevention: Review date on all materials
```
