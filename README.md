# AI-Powered K3 Health Promotion Media

> Leveraging AI tools (NotebookLM + Canva + Sora) to create professional occupational health and safety communication materials

[![AI Tools](https://img.shields.io/badge/AI-NotebookLM%20%7C%20Canva%20%7C%20Sora-blue)](https://github.com/raharjoo91/use-ai-to-create-media)
[![K3](https://img.shields.io/badge/Focus-K3%2FK3-green)](https://github.com/raharjoo91/use-ai-to-create-media)

## 📋 Overview

This open-source project provides practical guides, templates, and workflows for using AI tools to create effective health promotion media for workplace safety (K3 - Keselamatan dan Kesehatan Kerja).

### 🎯 What You'll Learn

- How to use **NotebookLM** to synthesize K3 research and generate content
- How to use **Canva AI** to design professional posters and infographics
- How to use **Sora** (when available) to generate safety training videos
- Complete workflows from research to final media
- Best practices for accessible, inclusive K3 communications

### 👥 Target Audience

- K3 professionals and safety officers
- Occupational health nurses
- Health promotion practitioners
- Public health students
- Workplace safety trainers
- Anyone creating K3 communication materials

## 🚀 Quick Start

### Prerequisites

```
✓ Google Account (for NotebookLM)
✓ Canva Account (free tier works)
✓ Basic understanding of K3 principles
✓ Internet connection
```

### 5-Minute Workflow

```
K3 Research → NotebookLM → Content + FAQs → Canva AI → Poster/Infographic → Test & Deploy
```

**Step 1:** Upload K3 guidelines to NotebookLM → Generate key points
**Step 2:** Use Canva Magic Design with key points → Create poster
**Step 3:** Review, test with users, deploy

## 🛠️ AI Tools Covered

### 1. NotebookLM (Google)

**Best for:**
- Research synthesis
- FAQ generation
- Content summarization
- Citation tracking

**Limitations:**
- File size: 50MB per source
- Source types: PDF, text, websites
- Language: Primarily English

### 2. Canva AI

**Best for:**
- Poster design
- Infographics
- Social media graphics
- Brand consistency

**Features:**
- Magic Design
- Text to Image
- Magic Edit
- Brand Kit

### 3. Sora (OpenAI)

**Best for:**
- Safety training videos
- Scenario demonstrations
- Emergency procedure visualization

**Status:** Check availability in your region
